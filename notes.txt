Создаем виртуальное окружение python 3.11
Устанавливаем зависимости из - requirements.txt
Устанавливаем сервер PostgreSQL 16 (пароль root)
Если необходимо устанавливаем нужные дополнения (PL/Python) в PostgreSQL 16
Запускаем на исполнение - test_plpython.py
В терминале запускаем приложение из файла app_fast_api.py, командой - uvicorn app_fast_api:app --reload
(находиться нужно в корне проекта при запуске)

Далее запускаем на исполнение файл start.py

Таким образом файл start.py в цикле пройдет по всем полученным id из таблицы users бд test_a
Передаст в внешний API id
API получит json от функции хранимой в бд test_a по id
При успешном получении отправит ответ 200 (ok) и запишет полученные данные в промежуточную таблицу бд test_b
